# Keyboards

Two types of keyboards for adding buttons to messages: reply keyboards (below input field) and inline keyboards (attached to messages).

## Quick Reference

| Need to... | Key Type(s) | File |
|------------|-------------|------|
| Custom keyboard below input field | ReplyKeyboardMarkup, KeyboardButton | reply-keyboards.md |
| Buttons attached to messages | InlineKeyboardMarkup, InlineKeyboardButton | inline-keyboards.md |
